    Extract position use a name pattern.

    Extract position use a name pattern.

    pattern

    Name pattern of the new coordinate. Use %[12]$s for the string representations of the left and right selected positions respectively.

    all

    Indicates if all positions should be returned (true), or only if reml is equal to remr.

    separator

    Separator to use when converting left and right positions to string.

    Note

    If a position is returned then it's always reml with an additional coordinate prepended.
