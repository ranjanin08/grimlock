    Note

    An Option is used in the return type to allow reducers to be selective in what contents.Content they apply to. For example, computing the mean is undefined for categorical variables. The reducer now has the option to return None. This in turn permits an external API, for simple cases, where the user need not know about the types of variables of their data.
